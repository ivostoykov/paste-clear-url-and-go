# Privacy Policy

**Paste Clear URL And Go**

## Data collection

This extension does not collect, transmit, or share any personal data.

All processing happens locally in your browser.

## Permissions used

| Permission | Why it is needed |
|------------|-----------------|
| `clipboardRead` | Read a URL from the clipboard when the popup is opened |
| `storage` | Save your settings (keep `id` parameter on/off) locally in the browser |
| `tabs` | Open the clean URL in the current or a new tab |
| `windows` | Open the clean URL in a new incognito/private window |
| `contextMenus` | Add menu items when right-clicking a link on a page |

## What stays on your device

- The **setting** (keep `id` parameter) is stored using the browser's built-in sync storage. It is synced across your own signed-in browser profiles only.
- **Clipboard content** is read only when you open the popup. It is never stored or sent anywhere.
- **Link URLs** from the right-click menu are processed in memory and discarded immediately after navigation.

## Third parties

No data is sent to any third-party server. The extension has no analytics, no telemetry, and no remote configuration.
