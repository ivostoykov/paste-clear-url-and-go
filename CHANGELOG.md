# Changelog

## 1.1.11

- Context menu on links: right-click any link to open it clean in current tab, new tab, or incognito tab
- Settings page title now shows the current version
- Fixed Settings link in popup not opening the options page

## 1.0.0

- Initial release
- Popup reads URL from clipboard and strips tracking query parameters
- Keeps `id` query parameter (case-insensitive) when setting is enabled
- Three actions: paste in current tab, new tab, or new incognito tab
- Settings page to toggle `id` parameter behaviour
- Chrome (MV3) and Firefox (MV3) support
